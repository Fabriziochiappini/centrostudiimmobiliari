## Come Caricare

1. Apri la sidebar di Replit
2. Trova il pannello "Object Storage"
3. Vai nella directory `public`
4. Carica i file con i nomi esatti indicati sopra

## Sistema Fallback

Se un file non viene trovato, il sito mostrerà automaticamente un placeholder con il nome del file da caricare, così saprai sempre quali file mancano.

Il sistema è già configurato e funzionante. URL serviti: `/public-objects/nome-file.jpg`
